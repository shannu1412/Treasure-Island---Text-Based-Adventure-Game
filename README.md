# Treasure Island - Text-Based Adventure Game

## 📌 Project Overview
Treasure Island is a simple text-based adventure game developed in Python. The player makes a series of choices to navigate through different paths in search of hidden treasure. The game demonstrates the use of conditional statements, loops, and user input handling to create an interactive experience.

## 🎯 Objective
The objective of the game is to find the treasure by making the correct decisions at each stage. Wrong choices lead to a Game Over, while the correct sequence leads to victory.

## 🛠 Technologies Used
- Python 3

## ✨ Features
- Interactive command-line gameplay
- Multiple decision-based paths
- Win and Game Over scenarios
- User-friendly text prompts
- "Try Again" option after game completion

## 📚 Concepts Used
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- User Input (`input()`)
- Functions
- String Manipulation

## 🎮 How to Play
1. Run the Python file.
2. Follow the on-screen instructions.
3. Enter your choices when prompted.
4. Reach the treasure to win the game.
5. Choose "Try Again" to restart after the game ends.

## ▶️ Sample Gameplay

```text
Welcome to Treasure Island!

You are at a crossroads.
Type 'left' or 'right': left

You come to a lake.
Type 'wait' or 'swim': wait

There are three doors: red, blue, yellow.
Choose a door: yellow

Congratulations! You found the treasure!
```
